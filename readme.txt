simple question/answers plugin. uses jquery-ui for effects.

== Description ==

I've created this plugin based on a [wptuts tutorial](http://wp.tutsplus.com/tutorials/creative-coding/create-an-faq-accordion-for-wordpress-with-jquery-ui/)
write your question and answers from the FAQ section in the admin panel and show them in your pages/posts using [wpfaq] shortcode or the relative widget.
 
== Installation ==

1. Upload WPFaqJquery directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
